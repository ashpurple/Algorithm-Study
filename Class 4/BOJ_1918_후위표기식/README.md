# Stack

Stack을 이용해 Post Prefix를 구현하는 문젱이다<br>
알고리즘은 다음과 같다<br>

1. 피연산자일 경우 무조건 결과에 넣기
2. 연산자일 경우
    - ( 일경우 스택에 넣기
    - ) 일경우 ( 가 나올 때 까지 결과에 넣으면서 pop
    - 사칙 연산일 경우 우선순위가 낮지 않을 때 top을 결과에 넣고 pop
    - 반복문을 다 탐색한 후 스택을 비워주며 결과값에 넣기

우선순위: */ 가 +-보다 높다.

즉 스택에 *가 있는데 +가 들어올 경우 *를 pop 하고 +를 push한다
